📝 ブラウザサポート状況

View Transition APIはモダンブラウザでサポートされています：

  • Chrome 111+ ✅
  • Edge 111+ ✅
  • Safari 18+ ✅
  • Firefox 実験的サポート 🚧

サポートされていないブラウザでは通常のページ遷移が行われます。

🔧 開発モードでの確認方法

以下のコマンドでアプリケーションを起動して画面遷移を確認：

npm run dev
# または
npm run build && npm run start
